### Options


<dl class="flags">
	<dt><code>--environment &lt;string&gt;</code></dt>
	<dd>Add/override environment variables during pull. Commas and equals are not allowed in the names or values.</dd>

	<dt><code>-y</code>, <code>--skip-confirmation</code></dt>
	<dd>Skip confirmation step</dd>

	<dt><code>--skip-db</code></dt>
	<dd>Skip pulling database archive</dd>

	<dt><code>--skip-files</code></dt>
	<dd>Skip pulling file archive</dd>

	<dt><code>--skip-import</code></dt>
	<dd>Downloads file and/or database archives, but does not import them</dd>
</dl>


### Options inherited from parent commands


<dl class="flags">
	<dt><code>-j</code>, <code>--json-output</code></dt>
	<dd>If true, user-oriented output will be in JSON format.</dd>
</dl>
