DDEV router received this request, but has nowhere to send it

This page is served directly by ddev-router (Traefik). It means the request reached the router, but the router has no project configured to handle this hostname/port. This is not a 404 from your project's web application.

A few things to check:

  • Is the project started? Run ddev start in the project directory.
  • Does the hostname you used match the project's configured hostnames? Run ddev describe in the project directory to see the URLs it responds to.
  • Is there a typo in the URL you used?

If none of that helps, see the DDEV documentation or ask in the DDEV Discord.
